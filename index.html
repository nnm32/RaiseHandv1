RaiseHand

Algebra confidence prototype

Practice quietly. Speak confidently.

RaiseHand helps a student practice algebra without being handed the answer. It gives hints, asks for teach-back, and rehearses the moment of raising a hand in class.

What this is doing

This is not an answer machine. It is a confidence bridge. The student practices thinking first, then practices saying the answer clearly.

1. Start with fear: name what feels confusing.
2. Hints before answers: guide thinking without solving everything.
3. Teach it back: explain the concept in your own words.
4. Raise-hand rehearsal: practice what you would say in class.
